## Instalacion :warning:

### :running: Ejecutar Local

Clonar el proyecto

```sh
- git clone https://github.com/jeroalderete/Black-Records
```

Ir a la carpeta del proyecto

```bash
  cd Black-Records
```

Instalar dependencias

```bash
  npm install
```

Start server

```bash
  npm run watch-css
```
